Tujuan: Meningkatkan aplikasi simulasi saham (https://github.com/kuriakinzeng/vuex-stock) dengan menambahkan kotak
sistem dan fitur drag-and-drop untuk membeli / menjual saham.

Cerita Pengguna:
# 1. Sebagai pengguna, saya dapat melihat Portofolio atau Saham masing-masing dengan sendirinya
# 2. Sebagai pengguna, saya bisa membuka grid baru sehingga saya bisa melihat Portofolio dan Saham bersama (dashboard)
# 3. Sebagai pengguna, saya bisa melihat daftar saham (dan harganya) yang saya bisa beli di bawah Stocks
# 4. Sebagai pengguna, saya bisa melihat daftar saham yang saya miliki dan nilainya (ini adalah portofolio)
5. Sebagai pengguna, saya bisa menarik saham dari jaringan Saham ke jaringan Portofolio untuk membeli Stok
6. Sebagai pengguna, saya dapat menarik saham dari jaringan Portofolio ke jaringan Saham untuk dijual Stok
7. Sebagai pengguna, saya bisa mensimulasikan pergerakan harga saham dan melihat nilai portofolio baru saya Dengan mengklik End Day
# 8. Sebagai pengguna, saya dapat menyimpan portofolio saya dan memuatnya kembali setelah refresh
9. Sebagai pengguna, saya bisa melihat tweet tentang setiap saham di daftar saham I Bisa beli

Bonus: Terapkan fitur tambahan yang akan membuat UX lebih baik. Marilah kita tahu apa itu meskipun!
